# Sample WhatsApp Chat Exports

Demo/testing chats for Ditto. Each file contains a distinct persona with a rich communication style.

## Files & Personas

| File | Persona to select | Style | Messages |
|------|------------------|-------|----------|
| `alex_dev.txt` | `Alex` | Software developer — opinionated, technical, dry humor | ~80 |
| `maya_creative.txt` | `Maya` | Visual artist — emotionally expressive, introspective | ~80 |
| `carlos_fitness.txt` | `Carlos` | Runner/fitness enthusiast — systematic, coaching, data-driven | ~80 |
| `priya_product.txt` | `Priya` | Product manager — frameworks-first, collaborative, sharp | ~80 |
| `leo_casual.txt` | `Leo` | Everyday person — curious, funny, self-aware | ~80 |

## How to use

1. Start Ditto (`bun run dev`)
2. Upload one of these `.txt` files
3. When prompted for the persona name, type the name from the **"Persona to select"** column above (exact case matters)
4. Chat with the generated twin

## Format

Standard WhatsApp export format:
```
DD/MM/YYYY, HH:MM AM/PM - Author: Message
```
